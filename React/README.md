# 安装说明

安装 cli

	sudo npm install -g create-react-app

创建新的项目

	create-react-app helloworld

运行项目

	npm run start

生产环境

	npm run build


本地服务器：

	npm install -g pushstate-server
	pushstate-server build
	open http://localhost:9000
